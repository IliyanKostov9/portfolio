\textbf{Project 2: Customer Data Insights \& Loyalty Programs (Whole Foods)} \\
\textit{Role: Data Engineer (Azure)} \\[0.5ex]

To support validation workflows, I co-developed a pytest-based Test Automation Framework for schema, duplication, data quality, and structural checks. The pipeline executed end-to-end validation, including:

\begin{itemize}

  \item Producing \texttt{junitxml\_report.xml} test artifacts for automated reporting.
\end{itemize}
